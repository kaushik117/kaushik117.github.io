# Kaushik Patel Portfolio Site

A single-page GitHub Pages portfolio site inspired by the structure of the reference site, but rewritten for Kaushik Patel's profile and experience.

## Files

- `index.html` — main page
- `style.css` — styles, theme, layout, responsive behavior
- `script.js` — theme toggle, mobile nav, scroll animation

## How to publish on GitHub Pages

1. Create a repository named `<your-github-username>.github.io`
2. Upload these files to the root of that repository
3. Replace `YOUR_GITHUB_USERNAME` in `index.html`
4. Commit and push to the `main` branch
5. Open `https://<your-github-username>.github.io`
